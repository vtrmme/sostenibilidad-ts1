# Proyecto sostenible con React, Typescript y bootstrap
vite: herramienta de construcción (build tool) moderna y ultra rápida para el desarrollo web frontend
nodejs: entorno de ejecución de código abierto y multiplataforma para JavaScript
react: biblioteca de JavaScript de código abierto, mantenida por Facebook y la comunidad, diseñada para crear interfaces de usuario (UI) interactivas y declarativas, principalmente para aplicaciones web de una sola página (SPAs).
